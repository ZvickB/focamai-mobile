# Focamai Designer Handoff

This bundle contains the current Focamai brand assets, product and design references, representative UI source, and a fresh Play Store feature-graphic brief.

## Start here

Read `FOCAMAI-BRAND-AND-FEATURE-GRAPHIC-BRIEF.md` first.

## Contents

- `brand-assets/` — primary wordmark, vector logo sources, app icon, and adaptive icon foreground.
- `brand-references/` — current mobile design system, mobile UI direction, and canonical web product notes.
- `ui-source/mobile/` — source for the current Search, Refine, Results, Detail, progress, theme, and shared surface system.
- `ui-source/web/` — source showing the web product experience and brand baseline.
- `fresh-screen-captures/` — intentionally empty placeholder for captures from the current running app.

## Important exclusions

- No file from `store-listing/` is included.
- The existing Play Store feature graphic was not reviewed or included.
- Archived UI mockups and prior promotional concepts are not included.
- No old screenshot is presented as current product UI.

## Screenshot capture request

Capture these screens from the current app at high resolution, preferably using one consistent Android device size:

1. Search — empty input state.
2. Refine — three useful chips and a realistic note.
3. Results — six picks, with a strong selected result visible.
4. Product Detail — product hero plus “Why this pick” and “Worth knowing.”

Use realistic but non-sensitive product examples. Keep the same search session across Refine, Results, and Detail so the story remains coherent.
